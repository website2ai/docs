# website2ai Documentation

This repository contains the documentation for website2ai, built with [Mintlify](https://mintlify.com).

## Development

Install the Mintlify CLI:

```bash
npm i -g mintlify
```

Run the development server:

```bash
mintlify dev
```

The documentation will be available at `http://localhost:3000`.

## Deployment

This documentation is automatically deployed via Mintlify when changes are pushed to the main branch.

## Structure

- `mint.json` - Configuration file
- `introduction.mdx` - Home page
- `quickstart.mdx` - Getting started guide
- `development.mdx` - Development guide
- `essentials/` - Core documentation
- `api-reference/` - API documentation
