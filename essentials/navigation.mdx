## Navigation Structure

Navigation is configured in `mint.json`:

```json
{
  "navigation": [
    {
      "group": "Group Name",
      "pages": [
        "page1",
        "page2",
        "folder/page3"
      ]
    }
  ]
}
```

## Page Order

Pages appear in the order they're listed in the navigation array.

## Nested Pages

You can organize pages in folders:

```
docs/
  ├── introduction.mdx
  ├── folder/
  │   ├── page1.mdx
  │   └── page2.mdx
```

Reference them as `folder/page1` in navigation.

## Tabs

Add multiple tabs to your documentation:

```json
{
  "tabs": [
    {
      "name": "API Reference",
      "url": "api-reference"
    }
  ]
}
```
