## 3단계로 시작하기

<Steps>
  <Step title="가입하기">
    [website2ai.com](https://website2ai.com)으로 이동하여 **"Google로 로그인"**을 클릭하세요

    계정이 즉시 생성됩니다 - 이메일 인증이 필요 없습니다.
  </Step>

  <Step title="빌더 열기">
    **"새 프로젝트 만들기"**를 클릭하거나 빌더로 이동하세요
  </Step>

  <Step title="웹사이트 설명하기">
    AI에게 만들고 싶은 것을 말하고 실시간으로 웹사이트가 생성되는 것을 보세요!
  </Step>
</Steps>

## 첫 번째 웹사이트 만들기

### 1. 새 프로젝트 시작

빌더에서 채팅 인터페이스를 볼 수 있습니다. 첫 번째 프롬프트를 입력하세요:

<Tip>
**시도해 볼 예시 프롬프트:**
- "생산성 앱을 위한 랜딩 페이지 만들기"
- "다크 모드가 있는 할 일 앱 만들기"
- "히어로 섹션과 프로젝트 갤러리가 있는 포트폴리오 웹사이트 만들기"
- "이커머스 제품 페이지 만들기"
</Tip>

### 2. AI가 웹사이트를 만드는 과정 보기

엔터를 누르면 website2ai는:

<Steps>
  <Step title="구조 계획">
    AI가 요청을 분석하고 파일 구조를 계획합니다
  </Step>

  <Step title="코드 생성">
    실시간으로 파일이 하나씩 생성되는 것을 보세요
  </Step>

  <Step title="실시간 미리보기">
    빌드되는 동안 미리보기 패널에서 웹사이트를 확인하세요
  </Step>
</Steps>

복잡도에 따라 일반적으로 **30-60초**가 걸립니다.

### 3. 웹사이트 미리보기

웹사이트가 오른쪽의 **실시간 미리보기 패널**에 나타납니다:

- ✅ 완전히 상호작용 가능하고 기능적
- ✅ 반응형 디자인
- ✅ 모든 컴포넌트 작동

### 4. 변경하기

웹사이트를 수정하고 싶으신가요? AI와 대화하세요:

```
"연락처 폼 추가"
"색상을 파란색으로 변경"
"히어로 섹션을 더 크게"
"다크 모드 토글 추가"
```

각 변경마다 새 버전이 생성되며 - 언제든지 돌아갈 수 있습니다!

### 5. 코드 다운로드

**"다운로드"** 버튼을 클릭하여 받으세요:

- TypeScript를 사용한 완전한 React 프로젝트
- 모든 소스 파일
- 종속성이 포함된 package.json
- 로컬에서 실행하거나 배포할 준비 완료

## 인터페이스 이해하기

### 왼쪽 패널: 채팅 및 코드

- **채팅 입력**: 원하는 것을 설명하거나 변경 요청
- **파일 트리**: 생성된 모든 파일 탐색
- **코드 에디터**: 파일의 코드 보기 및 복사

### 오른쪽 패널: 실시간 미리보기

- **미리보기 창**: 실시간으로 실행되는 웹사이트 보기
- **반응형 토글**: 모바일/태블릿/데스크톱 뷰 테스트
- **새로고침**: 필요시 미리보기 다시 로드

### 상단 바

- **프로젝트 제목**: 클릭하여 이름 변경
- **버전 드롭다운**: 다양한 버전 간 전환
- **다운로드 버튼**: 완전한 React 프로젝트 받기

## 프로젝트 관리

### 모든 프로젝트 보기

**"내 프로젝트"**를 클릭하여 모든 웹사이트를 확인하세요:

- 프로젝트 썸네일 및 제목 보기
- 상태별 필터링 (완료, 진행 중)
- 생성 또는 업데이트 날짜별 정렬

### 버전 히스토리

모든 변경은 새 버전을 생성합니다:

1. **버전 드롭다운**을 클릭
2. 프롬프트와 함께 모든 버전 보기
3. 버전을 클릭하여 보거나 복원

<Tip>
버전은 자동으로 번호가 매겨집니다: v1, v2, v3...

각 버전은 완전한 스냅샷입니다 - 언제든지 돌아갈 수 있습니다!
</Tip>

## 예시: 랜딩 페이지 만들기

완전한 랜딩 페이지를 만드는 과정을 살펴봅시다:

<Steps>
  <Step title="초기 프롬프트">
    ```
    "TaskFlow"라는 생산성 앱을 위한 현대적인 랜딩 페이지 만들기
    히어로 섹션, 기능 섹션, CTA 포함
    ```

    **결과:** 기본 랜딩 페이지 구조 (v1)
  </Step>

  <Step title="세부 정보 추가">
    ```
    Free, Pro, Enterprise의 3가지 등급이 있는 가격 섹션 추가
    ```

    **결과:** 가격이 포함된 랜딩 페이지 (v2)
  </Step>

  <Step title="디자인 다듬기">
    ```
    그라데이션과 블러 효과를 사용하여 디자인을 더 현대적으로
    ```

    **결과:** 향상된 시각적 디자인 (v3)
  </Step>

  <Step title="상호작용 추가">
    ```
    섹션이 화면에 나타날 때 부드러운 스크롤 애니메이션 추가
    ```

    **결과:** 애니메이션이 있는 랜딩 페이지 (v4)
  </Step>

  <Step title="다운로드">
    **다운로드**를 클릭하면 완전하고 운영 가능한 React 앱을 받게 됩니다!
  </Step>
</Steps>

    - 간단한 페이지: ~30초
    - 중간 복잡도: ~45초
    - 복잡한 앱: ~60-90초

  <Accordion title="코드를 직접 편집할 수 있나요?">

  </Accordion>

  <Accordion title="이전 버전은 어떻게 되나요?">
