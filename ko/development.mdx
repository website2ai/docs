## 일반 질문

<AccordionGroup>
  <Accordion title="website2ai는 무료인가요?">
    월별 제한된 생성 횟수를 제공하는 무료 등급이 있습니다. 자세한 내용은 [가격 페이지](https://website2ai.com/pricing)를 확인하세요.
  </Accordion>

  <Accordion title="website2ai로 무엇을 만들 수 있나요?">
    다음을 만들 수 있습니다:
    - 랜딩 페이지
    - 웹 애플리케이션
    - 대시보드
    - 이커머스 사이트
    - 포트폴리오
    - 블로그
    - 문서 사이트

    기본적으로 모든 React 기반 웹사이트를 만들 수 있습니다!
  </Accordion>

  <Accordion title="생성된 코드를 소유하나요?">
    **네!** website2ai가 생성한 모든 코드는 100% 귀하의 것입니다. 상업적으로 사용하고, 수정하고, 판매하세요 - 원하는 대로 하세요.
  </Accordion>

  <Accordion title="코드를 편집할 수 있나요?">
    물론입니다! 프로젝트를 다운로드하고 일반 React 앱처럼 편집하세요. 코드는 깔끔하고 읽기 쉬우며 모범 사례를 따릅니다.
  </Accordion>
</AccordionGroup>

## 기술적 질문

<AccordionGroup>
  <Accordion title="website2ai는 어떤 기술을 사용하나요?">
    생성된 웹사이트는 다음을 사용합니다:
    - **React** - UI 프레임워크
    - **TypeScript** - 타입 안정성
    - **Tailwind CSS** - 스타일링
    - **Vite** - 빌드 도구

    모두 현대적이고 운영 가능한 기술입니다!
  </Accordion>

  <Accordion title="생성하는 데 얼마나 걸리나요?">
    일반적인 생성 시간:
    - 간단한 페이지: ~30초
    - 중간 복잡도: ~45초
    - 복잡한 앱: ~60-90초
  </Accordion>

  <Accordion title="자체 디자인 시스템을 사용할 수 있나요?">
    프롬프트에서 디자인 선호도를 설명할 수 있습니다:

    "파란색 기본 색상과 주황색 강조를 사용한 색상 구성표 사용"

    "Material Design 원칙 따르기"

    "Stripe와 같은 카드 기반 레이아웃 사용"

    완전한 맞춤 디자인 시스템의 경우 코드를 다운로드하고 수정하세요.
  </Accordion>

  <Accordion title="백엔드 API와 작동하나요?">
    네! API 통합을 요청할 수 있습니다:

    "/api/todos의 REST API에 연결하는 할 일 앱 만들기"

    생성된 코드에는 API 호출 및 상태 관리가 포함됩니다.
  </Accordion>
</AccordionGroup>

## 계정 및 결제

<AccordionGroup>
  <Accordion title="요금제를 업그레이드하려면 어떻게 하나요?">
    [계정 설정](https://website2ai.com/settings) → 결제 → 요금제 선택으로 이동하세요
  </Accordion>

  <Accordion title="언제든지 취소할 수 있나요?">
    네, 언제든지 위약금 없이 취소할 수 있습니다. 청구 기간이 끝날 때까지 액세스가 계속됩니다.
  </Accordion>

  <Accordion title="어떤 결제 방법을 사용할 수 있나요?">
    Stripe를 통해 모든 주요 신용카드를 사용할 수 있습니다.
  </Accordion>

  <Accordion title="환불을 제공하나요?">
    모든 유료 요금제에 대해 14일 환불 보증을 제공합니다.
  </Accordion>
</AccordionGroup>
