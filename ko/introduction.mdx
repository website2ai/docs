## website2ai에 오신 것을 환영합니다

**website2ai**는 여러분의 아이디어를 실제 운영 가능한 React 코드로 변환하는 AI 기반 웹사이트 빌더입니다. 원하는 것을 설명하기만 하면 AI가 몇 분 안에 완전히 작동하는 웹사이트를 만들어냅니다.

### 무엇을 만들 수 있나요?

- **랜딩 페이지**: 마케팅 사이트, 제품 페이지, 포트폴리오
- **웹 애플리케이션**: 할 일 앱, 대시보드, 관리자 패널
- **이커머스**: 제품 카탈로그, 장바구니, 결제 플로우
- **비즈니스 사이트**: 기업 웹사이트, 블로그, 문서

### 주요 기능

- **AI 기반 생성**: 평범한 한국어로 비전을 설명하면 실제 운영 가능한 코드를 받습니다
- **실시간 미리보기**: 웹사이트가 생성되는 동안 실시간으로 확인하세요
- **완전한 코드 접근**: TypeScript를 사용한 완전한 React 프로젝트 다운로드
- **버전 히스토리**: 모든 반복 작업을 추적하고 이전 버전으로 돌아가기
- **즉각적인 업데이트**: AI와 대화하여 변경 - "다크 모드 추가", "더 화려하게"
- **실시간 진행 상황**: 파일별 생성 진행 상황 확인

## 이런 분들께 완벽합니다

- **창업가**: 랜딩 페이지와 MVP를 빠르게 프로토타입
- **개발자**: 보일러플레이트 코드 생성 및 개발 속도 향상
- **디자이너**: 디자인을 작동하는 React 코드로 변환
- **에이전시**: 고객 프레젠테이션을 위한 빠른 프로토타입
- **학생**: AI가 완전한 프로젝트를 만드는 과정을 보며 React 배우기

## 무엇을 받게 되나요?

생성된 모든 웹사이트에는 다음이 포함됩니다:

- ✅ TypeScript를 사용한 완전한 React 프로젝트
- ✅ Tailwind CSS를 사용한 반응형 디자인
- ✅ 현대적인 UI 컴포넌트
- ✅ 깔끔하고 읽기 쉬운 코드
- ✅ 다운로드 가능한 모든 소스 파일
- ✅ 실시간 미리보기 환경
