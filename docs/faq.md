# Frequently Asked Questions

## General

### What is Website2AI?

Website2AI is an AI-powered platform that creates websites from natural language descriptions. You describe what you want, and our AI generates a complete, production-ready website using React, TypeScript, and Tailwind CSS.

### Do I need to know how to code?

No. Website2AI is designed for everyone. Describe your website in plain English, and the AI handles all the technical work. If you do know how to code, you can view and edit the generated source code.

### What kind of websites can I create?

You can create:
- Landing pages
- Portfolio websites
- Business websites
- Blog layouts
- Dashboard interfaces
- E-commerce storefronts
- Web applications
- Documentation sites

### What technologies are used?

Generated websites use:
- **React 18+** — Modern component-based UI
- **TypeScript** — Type-safe, maintainable code
- **Tailwind CSS** — Utility-first styling
- **Vite** — Fast build tooling

Websites are hosted on Cloudflare's global edge network.

## Account & Credits

### How do I sign up?

Visit [website2.ai](https://website2.ai) and sign in with your Google account.

### Is there a free plan?

Yes. Free accounts receive signup bonus credits (permanent) and daily credits (expire after 24 hours).

### How do credits work?

Credits are consumed based on AI usage. Simple changes cost fewer credits; complex generations cost more. The exact cost depends on the complexity of your request and the AI resources used.

### Do credits expire?

- **Signup and purchased credits** — Never expire
- **Monthly credits** — 1 month carryover (unused credits expire the following month)
- **Daily credits** — Expire at midnight UTC

Credits are consumed in order of expiration date.

### What's included in the Pro plan?

Pro plan includes:
- Higher signup bonus
- Monthly credit grants (1 month carryover)
- AI image generation
- Custom domains
- Remove "Powered by" badge
- Private previews *(coming soon)*
- Priority support

Visit [website2.ai/pricing](https://website2.ai/pricing) for current pricing.

### How do I upgrade to Pro?

1. Click your profile icon
2. Go to **Account** → **Subscription**
3. Select **Pro** and complete payment

## Website Creation

### How long does generation take?

Most websites generate in 1-3 minutes, depending on complexity. You'll see real-time progress as files are created.

### Can I edit my website after it's created?

Yes. Use the chat to request changes like:
- "Change the header color to blue"
- "Add a contact form section"
- "Make the buttons rounded"

### Can I undo changes?

Version history with restore functionality is coming soon. Once available, you'll be able to browse all previous versions and restore any earlier version.

### Can I view the source code?

Yes. Toggle the code panel to see all files, including React components, TypeScript code, and Tailwind styles.

### Can I download the code?

Code download is coming soon.

### How many websites can I create?

Unlimited. Each generation uses credits based on complexity.

### Can I delete a website?

Yes. Delete projects from your dashboard. Deleted projects cannot be recovered.

## Domains & Hosting

### Where are websites hosted?

On Cloudflare's global edge network, ensuring fast load times worldwide.

### What URL will my website have?

Default: `https://your-project.website2.ai`

Pro plan can connect custom domains.

### Can I use my own domain?

Yes, with Pro plan. See [Custom Domains](/custom-domains) for setup.

### Is HTTPS included?

Yes. All websites use HTTPS with automatic SSL certificates.

### Is hosting included?

Yes. Hosting is included for all plans. Websites remain live while your account is active.

## Technical

### Are websites mobile-responsive?

Yes. All generated websites are responsive by default. You can preview mobile and desktop layouts in the builder.

### What browsers are supported?

Generated websites work on:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers

### Can I add a database?

Currently, Website2AI generates frontend code. Database integration is planned for future releases.

### Can I add forms that submit data?

The AI generates form UI. For form submissions, you'll need to integrate with a backend service like Formspree or your own API.

### Is there an API?

API access is coming soon. Contact support@website2.ai for updates.

## Privacy & Security

### Who owns the generated code?

You own all code generated for your projects. Use, modify, and distribute it as you wish.

### Is my data secure?

Yes. We use encryption for all data in transit and at rest. Projects are private by default.

### Are projects private?

Yes, by default. You control who can view your projects.

### Do you use my data to train AI?

No. Your code and prompts are not used to train AI models without explicit consent.

## Troubleshooting

### Generation is taking too long

Try:
- Simplifying your request
- Breaking it into smaller parts
- Checking your internet connection

### The result doesn't match my description

Try:
- Being more specific
- Listing sections explicitly
- Including style details (colors, spacing, fonts)

### Build failed

The AI automatically attempts to fix build errors. If issues persist:
- Request a simpler version
- Start fresh with a new generation
- Contact support if repeated failures

### Preview not loading

Try:
- Refreshing the preview
- Clearing your browser cache
- Trying a different browser
