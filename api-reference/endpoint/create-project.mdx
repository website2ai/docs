---

## Overview

Creates a new website project with version 1. The project is created in `pending` state and must be generated using the generate endpoint.

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer JWT token for authentication
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

## Body Parameters

<ParamField body="initial_prompt" type="string" required>
  The initial prompt describing what website to build. This will be stored as the first message.
</ParamField>

<ParamField body="title" type="string">
  Optional human-readable title for the project. If not provided, it will be generated from the prompt.
</ParamField>

## Response

<ResponseField name="base_id" type="string">
  Unique identifier for the project (shared across all versions)
</ResponseField>

<ResponseField name="version" type="number">
  Version number (always 1 for new projects)
</ResponseField>

<ResponseField name="website_id" type="string">
  Unique identifier for this specific version (base_id + version)
</ResponseField>

<ResponseField name="status" type="string">
  Status of the website (`pending`, `processing`, `completed`, `failed`)
</ResponseField>

<ResponseField name="title" type="string">
  Project title
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of creation
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp of last update
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X POST http://localhost:8000/v1/websites \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "initial_prompt": "Create a modern landing page for a SaaS product with hero section, features, and pricing",
    "title": "SaaS Landing Page"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:8000/v1/websites', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    initial_prompt: 'Create a modern landing page for a SaaS product with hero section, features, and pricing',
    title: 'SaaS Landing Page'
  })
});

const data = await response.json();
console.log('Project created:', data.base_id);
```

```python Python
import requests

response = requests.post(
    'http://localhost:8000/v1/websites',
    headers={'Authorization': 'Bearer YOUR_JWT_TOKEN'},
    json={
        'initial_prompt': 'Create a modern landing page for a SaaS product with hero section, features, and pricing',
        'title': 'SaaS Landing Page'
    }
)

data = response.json()
print(f"Project created: {data['base_id']}")
```

</CodeGroup>

## Next Steps

After creating a project, you should:

1. **Generate the website**: Call `POST /v1/websites/{base_id}/generate` to start code generation
2. **Stream progress**: Connect to SSE endpoint to watch real-time updates
3. **Retrieve files**: Use `GET /v1/websites/{base_id}` to get the generated code
