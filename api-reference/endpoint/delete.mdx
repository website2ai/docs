---
title: 'Delete Content'
api: 'DELETE https://api.website2ai.com/v1/content/{id}'
description: 'Delete processed content'
---

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the content to delete
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the deletion was successful
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X DELETE https://api.website2ai.com/v1/content/abc123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript Node.js
const response = await fetch('https://api.website2ai.com/v1/content/abc123', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});
const data = await response.json();
```

```python Python
import requests

response = requests.delete(
  'https://api.website2ai.com/v1/content/abc123',
  headers={'Authorization': 'Bearer YOUR_API_KEY'}
)
data = response.json()
```

</CodeGroup>

## Response Example

```json
{
  "success": true,
  "message": "Content successfully deleted"
}
```

## Error Responses

### Content Not Found

```json
{
  "success": false,
  "error": {
    "code": "NOT_FOUND",
    "message": "Content with ID abc123 not found"
  }
}
```
