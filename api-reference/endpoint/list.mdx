## Overview

Retrieves a list of all website projects for the authenticated user. Returns latest version of each project.

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer JWT token for authentication
</ParamField>

## Query Parameters

<ParamField query="skip" type="number" default="0">
  Number of records to skip for pagination
</ParamField>

<ParamField query="limit" type="number" default="20">
  Maximum number of records to return (max: 100)
</ParamField>

## Response

Returns an array of project objects:

<ResponseField name="base_id" type="string">
  Project base identifier
</ResponseField>

<ResponseField name="version" type="number">
  Latest version number
</ResponseField>

<ResponseField name="website_id" type="string">
  Unique identifier for latest version
</ResponseField>

<ResponseField name="status" type="string">
  Generation status of latest version
</ResponseField>

<ResponseField name="title" type="string">
  Project title
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp when project was created
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp of latest version update
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "http://localhost:8000/v1/websites?skip=0&limit=20" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```javascript JavaScript
const response = await fetch(
  'http://localhost:8000/v1/websites?skip=0&limit=20',
  {
    headers: {
      'Authorization': 'Bearer YOUR_JWT_TOKEN'
    }
  }
);

const projects = await response.json();

projects.forEach(project => {
  console.log(`${project.title} (v${project.version}): ${project.status}`);
});
```

```python Python
import requests

response = requests.get(
    'http://localhost:8000/v1/websites',
    headers={'Authorization': 'Bearer YOUR_JWT_TOKEN'},
    params={'skip': 0, 'limit': 20}
)

projects = response.json()

for project in projects:
    print(f"{project['title']} (v{project['version']}): {project['status']}")
```

</CodeGroup>

## Pagination

For large result sets, use pagination:

```javascript
// Get first page
let skip = 0;
const limit = 20;
const allProjects = [];

while (true) {
  const response = await fetch(
    `http://localhost:8000/v1/websites?skip=${skip}&limit=${limit}`,
    { headers: { 'Authorization': 'Bearer YOUR_JWT_TOKEN' } }
  );

  const projects = await response.json();
  if (projects.length === 0) break;

  allProjects.push(...projects);
  skip += limit;
}

console.log(`Total projects: ${allProjects.length}`);
```

## Filtering by Status

The API currently returns all projects. To filter by status on the client side:

```javascript
const response = await fetch('http://localhost:8000/v1/websites', {
  headers: { 'Authorization': 'Bearer YOUR_JWT_TOKEN' }
});

const projects = await response.json();

// Get only completed projects
const completed = projects.filter(p => p.status === 'completed');

// Get projects currently processing
const processing = projects.filter(p => p.status === 'processing');

// Get failed projects
const failed = projects.filter(p => p.status === 'failed');
```
