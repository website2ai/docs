## Welcome to the API

The website2ai API allows you to programmatically convert websites into AI-ready content.

## Authentication

All API requests require authentication using an API key:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.website2ai.com/v1/convert
```

## Base URL

```
https://api.website2ai.com/v1
```

## Rate Limits

- Free tier: 100 requests/day
- Pro tier: 10,000 requests/day
- Enterprise: Custom limits

## Response Format

All responses are returned in JSON format:

```json
{
  "success": true,
  "data": {
    // Response data
  }
}
```

## Error Handling

Errors are returned with appropriate HTTP status codes:

```json
{
  "success": false,
  "error": {
    "code": "ERROR_CODE",
    "message": "Error description"
  }
}
```
