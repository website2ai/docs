## Welcome to website2ai

**website2ai** is an AI-powered website builder that turns your ideas into production-ready React code. Simply describe what you want, and watch as AI creates a complete, working website in minutes.

### What You Can Build

- **Landing Pages**: Marketing sites, product pages, portfolios
- **Web Applications**: Todo apps, dashboards, admin panels
- **E-commerce**: Product catalogs, shopping carts, checkout flows
- **Business Sites**: Corporate websites, blogs, documentation

### Key Features

- **AI-Powered Generation**: Describe your vision in plain English, get production-ready code
- **Live Preview**: See your website come to life as it's being generated
- **Full Code Access**: Download complete React project with TypeScript
- **Version History**: Track every iteration and go back to any previous version
- **Instant Updates**: Make changes by chatting with AI - "add dark mode", "make it more colorful"
- **Real-time Progress**: Watch file-by-file generation progress
