## General Questions

<AccordionGroup>
  <Accordion title="Is website2ai free?">
    We offer a free tier with limited generations per month. Check our [pricing page](https://website2ai.com/pricing) for details.
  </Accordion>

  <Accordion title="What can I build with website2ai?">
    You can build:
    - Landing pages
    - Web applications
    - Dashboards
    - E-commerce sites
    - Portfolios
    - Blogs
    - Documentation sites

    Basically any React-based website!
  </Accordion>

  <Accordion title="Do I own the generated code?">
    **Yes!** All code generated by website2ai is 100% yours. Use it commercially, modify it, sell it - whatever you want.
  </Accordion>

  <Accordion title="Can I edit the code?">
    Absolutely! Download your project and edit it like any React app. The code is clean, readable, and follows best practices.
  </Accordion>
</AccordionGroup>

## Technical Questions

<AccordionGroup>
  <Accordion title="What technologies does website2ai use?">
    Generated websites use:
    - **React** - UI framework
    - **TypeScript** - Type safety
    - **Tailwind CSS** - Styling
    - **Vite** - Build tool

    All modern, production-ready technologies!
  </Accordion>

  <Accordion title="How long does generation take?">
    Typical generation times:
    - Simple pages: ~30 seconds
    - Medium complexity: ~45 seconds
    - Complex apps: ~60-90 seconds
  </Accordion>

  <Accordion title="Can I use my own design system?">
    You can describe your design preferences in prompts:

    "Use a color scheme of blue primary with orange accents"

    "Follow Material Design principles"

    "Use a card-based layout like Stripe"

    For complete custom design systems, download and modify the code.
  </Accordion>

  <Accordion title="Does it work with backend APIs?">
    Yes! You can request API integrations:

    "Create a todo app that connects to a REST API at /api/todos"

    The generated code will include API calls and state management.
  </Accordion>
</AccordionGroup>
