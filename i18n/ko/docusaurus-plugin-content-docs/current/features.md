# 기능

Website2AI가 제공하는 모든 것을 살펴보세요.

## AI 웹사이트 생성

### 자연어 입력

일반 언어로 웹사이트를 설명하세요. AI가 맥락, 스타일 선호도, 복잡한 요구 사항을 이해합니다.

**지원되는 웹사이트 유형:**
- 랜딩 페이지 및 마케팅 사이트
- 포트폴리오 및 개인 웹사이트
- 비즈니스 및 회사 웹사이트
- 블로그 레이아웃
- 대시보드 인터페이스
- 웹 애플리케이션
- 이커머스 스토어프론트
- 문서 사이트

### 실시간 생성

웹사이트가 탄생하는 것을 지켜보세요:

- **실시간 진행 업데이트** - 각 파일이 생성되는 것을 확인
- **즉시 미리보기** - 코드가 생성됨에 따라 실시간으로 미리보기 업데이트
- **빌드 상태** - 컴파일 및 배포 진행 상황 추적

### 프로덕션 수준 코드

Website2AI는 현대적이고 깔끔한 코드를 생성합니다:

- **React 18+** - 컴포넌트 기반 아키텍처
- **TypeScript** - 타입 안전하고 유지보수 가능한 코드
- **Tailwind CSS** - 유틸리티 우선 스타일링
- **Vite** - 빠른 빌드 도구
- **반응형 디자인** - 모바일 우선 접근 방식

### AI 모델

Website2AI는 웹사이트 생성을 위한 최신의 가장 능력 있는 모델인 **GPT-5.2**를 사용합니다. 복잡한 요구 사항을 이해하고 고품질의 프로덕션 수준 코드를 생성하는 데 탁월합니다.

## 채팅 기반 편집

### 반복적 수정

초기 웹사이트가 생성된 후 AI와 채팅하여 다듬으세요:

```
푸터에 뉴스레터 가입 양식을 추가해줘
```

```
히어로 섹션을 더 높게 하고 그라데이션 배경을 넣어줘
```

```
모든 버튼에 둥근 모서리와 호버 효과를 추가해줘
```

### 맥락 인식 변경

AI가 전체 프로젝트를 기억합니다. 이전 작업을 기반으로 변경을 요청하세요:

```
헤더에 사용한 것과 같은 색상 테마를 가격 섹션에 적용해줘
```

```
기존 것과 같은 후기 카드를 하나 더 추가해줘
```

## 버전 기록

### 자동 버전 관리

모든 수정은 새 버전을 생성합니다:

- **버전 추적** - 각 변경 사항이 새 버전으로 저장
- **기록 탐색** - 모든 이전 반복 확인
- **이전 버전 복원** - 어떤 이전 버전으로든 돌아가기

### 버전 세부 정보

각 버전에서 확인할 수 있는 것:
- 생성한 프롬프트
- 생성된 시간
- 전체 소스 코드
- 실시간 미리보기

## 배포 및 호스팅

### 즉시 배포

모든 웹사이트가 자동으로:

1. 프로덕션 최적화로 빌드됨
2. Cloudflare의 글로벌 엣지 네트워크에 배포됨
3. 고유 서브도메인으로 접근 가능

### 서브도메인 호스팅

웹사이트는 다음 주소에서 사용 가능:

```
https://your-project-name.website2.ai
```

기능:
- **글로벌 CDN** - 전 세계 빠른 로드 시간
- **SSL/HTTPS** - 기본적으로 보안
- **항상 사용 가능** - 99.9% 가동 시간

### 커스텀 도메인

자신의 도메인 연결 (Pro/Business 플랜):

- 전체 SSL/HTTPS 지원
- 자동 인증서 프로비저닝
- 간단한 A 레코드 구성

설정 지침은 [커스텀 도메인](/ko/custom-domains)을 참조하세요.

## AI 이미지 생성

웹사이트를 위한 커스텀 이미지 생성:

- **히어로 이미지** - 눈길을 끄는 배경
- **일러스트레이션** - 커스텀 그래픽
- **제품 샷** - 플레이스홀더 이미지
- **아이콘** - 커스텀 아이코노그래피

이미지 생성은 추가 크레딧을 사용합니다. 사용 기술:
- **GPT Image 1.5** - 고품질 사실적 이미지
- **Nano Banana Pro2** - 빠르고 비용 효율적인 생성

## 미리보기 모드

### 데스크톱 미리보기

전체 너비 레이아웃으로 데스크톱 화면에서 웹사이트가 어떻게 보이는지 확인하세요.

### 모바일 미리보기

모바일 뷰로 전환하여 반응형 디자인을 확인하고 모든 기기에서 사이트가 작동하는지 확인하세요.

### 전체 화면

새 탭에서 미리보기를 열어 에디터 인터페이스 없이 웹사이트를 확인하세요.

## 보안

### 인프라 보안

- **모든 곳에서 HTTPS** - 모든 트래픽 암호화
- **엣지 네트워크** - Cloudflare의 DDoS 보호
- **보안 스토리지** - 저장 시 암호화

### 코드 보안

- **서버 접근 없음** - 정적 사이트, 백엔드 취약점 없음
- **콘텐츠 보안 정책** - XSS 보호
- **보안 헤더** - 모범 사례 HTTP 헤더

### 데이터 프라이버시

- 기본적으로 비공개 프로젝트
- 동의 없이 AI 학습에 코드 사용 안 함
- GDPR 준수 데이터 처리
