# 자주 묻는 질문

## 일반

### Website2AI란 무엇인가요?

Website2AI는 자연어 설명으로 웹사이트를 만드는 AI 기반 플랫폼입니다. 원하는 것을 설명하면 AI가 React, TypeScript, Tailwind CSS를 사용하여 완전한 프로덕션 수준의 웹사이트를 생성합니다.

### 코딩을 알아야 하나요?

아니요. Website2AI는 모든 사람을 위해 설계되었습니다. 일반 언어로 웹사이트를 설명하면 AI가 모든 기술적인 작업을 처리합니다. 코딩을 안다면 생성된 소스 코드를 보고 편집할 수 있습니다.

### 어떤 종류의 웹사이트를 만들 수 있나요?

다음을 만들 수 있습니다:
- 랜딩 페이지
- 포트폴리오 웹사이트
- 비즈니스 웹사이트
- 블로그 레이아웃
- 대시보드 인터페이스
- 이커머스 스토어프론트
- 웹 애플리케이션
- 문서 사이트

### 어떤 기술이 사용되나요?

생성된 웹사이트는 다음을 사용합니다:
- **React 18+** — 모던 컴포넌트 기반 UI
- **TypeScript** — 타입 안전하고 유지보수 가능한 코드
- **Tailwind CSS** — 유틸리티 우선 스타일링
- **Vite** — 빠른 빌드 도구

웹사이트는 Cloudflare의 글로벌 엣지 네트워크에서 호스팅됩니다.

## 계정 및 크레딧

### 어떻게 가입하나요?

[website2.ai](https://website2.ai)를 방문하여 Google 계정으로 로그인하세요.

### 무료 플랜이 있나요?

예. 무료 계정은 가입 보너스 크레딧 (영구)과 일일 크레딧 (24시간 후 만료)을 받습니다.

### 크레딧은 어떻게 작동하나요?

크레딧은 AI 사용량에 따라 소비됩니다. 간단한 변경은 더 적은 크레딧을 소비하고 복잡한 생성은 더 많이 소비합니다. 정확한 비용은 요청의 복잡도와 사용된 AI 리소스에 따라 달라집니다.

### 크레딧은 만료되나요?

- **가입, 구매, 월간 크레딧** — 절대 만료되지 않음
- **일일 크레딧** — 자정 UTC에 만료

일일 크레딧이 먼저 소비되고, 그 다음 영구 크레딧이 소비됩니다.

### Pro 플랜에는 무엇이 포함되나요?

Pro 플랜 포함:
- 더 높은 가입 보너스
- 더 많은 일일 크레딧
- 월간 크레딧 지급 (영구)
- 커스텀 도메인
- 비공개 미리보기
- 우선 지원

현재 가격은 [website2.ai/pricing](https://website2.ai/pricing)을 방문하세요.

### Pro로 업그레이드하려면 어떻게 하나요?

1. 프로필 아이콘 클릭
2. **계정** → **구독** 이동
3. **Pro** 선택하고 결제 완료

## 웹사이트 생성

### 생성에 얼마나 걸리나요?

대부분의 웹사이트는 복잡도에 따라 1-3분 안에 생성됩니다. 파일이 생성되는 동안 실시간 진행 상황을 볼 수 있습니다.

### 생성 후 웹사이트를 편집할 수 있나요?

예. 채팅을 사용하여 다음과 같은 변경을 요청하세요:
- "헤더 색상을 파란색으로 변경해줘"
- "문의 양식 섹션 추가해줘"
- "버튼을 둥글게 만들어줘"

각 변경은 새 버전을 생성합니다.

### 변경을 취소할 수 있나요?

예. 모든 수정은 새 버전을 생성합니다. 버전 기록을 탐색하고 이전 버전을 복원할 수 있습니다.

### 소스 코드를 볼 수 있나요?

예. 코드 패널을 토글하여 React 컴포넌트, TypeScript 코드, Tailwind 스타일을 포함한 모든 파일을 확인할 수 있습니다.

### 코드를 다운로드할 수 있나요?

코드 다운로드는 Business 플랜 구독자에게 제공됩니다.

### 웹사이트를 몇 개나 만들 수 있나요?

무제한. 각 생성은 복잡도에 따라 크레딧을 사용합니다.

### 웹사이트를 삭제할 수 있나요?

예. 대시보드에서 프로젝트를 삭제할 수 있습니다. 삭제된 프로젝트는 복구할 수 없습니다.

## 도메인 및 호스팅

### 웹사이트는 어디서 호스팅되나요?

Cloudflare의 글로벌 엣지 네트워크에서 전 세계 빠른 로드 시간을 보장합니다.

### 웹사이트 URL은 무엇인가요?

기본값: `https://your-project.website2.ai`

Pro/Business 플랜은 커스텀 도메인을 연결할 수 있습니다.

### 제 도메인을 사용할 수 있나요?

예, Pro 또는 Business 플랜으로. 설정은 [커스텀 도메인](/ko/custom-domains)을 참조하세요.

### HTTPS가 포함되나요?

예. 모든 웹사이트는 자동 SSL 인증서와 함께 HTTPS를 사용합니다.

### 호스팅이 포함되나요?

예. 호스팅은 모든 플랜에 포함됩니다. 계정이 활성화되어 있는 동안 웹사이트는 라이브 상태를 유지합니다.

## 기술

### 웹사이트는 모바일 반응형인가요?

예. 모든 생성된 웹사이트는 기본적으로 반응형입니다. 빌더에서 모바일과 데스크톱 레이아웃을 미리 볼 수 있습니다.

### 어떤 브라우저가 지원되나요?

생성된 웹사이트는 다음에서 작동합니다:
- Chrome (최신)
- Firefox (최신)
- Safari (최신)
- Edge (최신)
- 모바일 브라우저

### 데이터베이스를 추가할 수 있나요?

현재 Website2AI는 프론트엔드 코드를 생성합니다. 데이터베이스 통합은 향후 릴리스에서 계획되어 있습니다.

### 데이터를 제출하는 양식을 추가할 수 있나요?

AI가 양식 UI를 생성합니다. 양식 제출의 경우 Formspree와 같은 백엔드 서비스 또는 자체 API와 통합해야 합니다.

### API가 있나요?

API 접근은 Business 플랜 구독자에게 제공됩니다. 자세한 내용은 support@website2.ai로 문의하세요.

## 개인정보 및 보안

### 생성된 코드는 누가 소유하나요?

프로젝트를 위해 생성된 모든 코드를 소유합니다. 원하는 대로 사용, 수정, 배포하세요.

### 데이터는 안전한가요?

예. 전송 중 및 저장 시 모든 데이터에 암호화를 사용합니다. 프로젝트는 기본적으로 비공개입니다.

### 프로젝트는 비공개인가요?

예, 기본적으로. 누가 프로젝트를 볼 수 있는지 제어합니다.

### AI 학습에 데이터를 사용하나요?

아니요. 명시적인 동의 없이 코드와 프롬프트를 AI 모델 학습에 사용하지 않습니다.

## 문제 해결

### 생성이 너무 오래 걸림

시도해보세요:
- 요청 단순화
- 더 작은 부분으로 나누기
- 인터넷 연결 확인

### 결과가 설명과 맞지 않음

시도해보세요:
- 더 구체적으로
- 섹션을 명시적으로 나열
- 스타일 세부 사항 포함 (색상, 간격, 글꼴)

### 빌드 실패

AI가 자동으로 빌드 오류 수정을 시도합니다. 문제가 지속되면:
- 더 단순한 버전 요청
- 새로운 생성으로 다시 시작
- 반복적인 실패 시 지원팀에 문의

### 미리보기가 로드되지 않음

시도해보세요:
- 미리보기 새로고침
- 브라우저 캐시 삭제
- 다른 브라우저 시도
