# 크레딧 및 가격

Website2AI는 플랫폼 사용 방식에 유연성을 제공하는 크레딧 기반 시스템을 사용합니다.

## 크레딧 작동 방식

크레딧은 Website2AI에서 웹사이트를 생성하고 수정하기 위한 통화입니다.

### 크레딧 소비

크레딧은 AI 사용량에 따라 소비됩니다. 비용은 다음에 따라 달라집니다:

- **복잡도** — 간단한 텍스트 변경은 전체 페이지 생성보다 적게 소비
- **AI 모델** — 다른 모델은 다른 비용을 가짐
- **이미지 생성** — AI 생성 이미지는 추가 크레딧 소비
- **토큰 사용량** — 더 복잡한 프롬프트와 출력은 더 많은 크레딧 사용

### 크레딧 유형

Website2AI에는 두 가지 유형의 크레딧이 있습니다:

| 유형 | 동작 |
|------|----------|
| **영구 크레딧** | 만료되지 않음. 가입 보너스, 구매에서 제공. |
| **월간 크레딧** | Pro 구독자에게 매월 지급. 미사용 시 1개월 이월 가능. |
| **일일 크레딧** | 24시간 후 만료. 매일 자정 UTC에 갱신. |

**사용 우선순위:** 일일 크레딧이 먼저 소비되고, 그 다음 영구 크레딧이 소비됩니다.

## 구독 플랜

### Free 플랜

무료로 시작하세요:

- 가입 보너스 크레딧 (영구)
- 일일 크레딧 (24시간 만료)
- 공개 미리보기
- 기본 기능

### Pro 플랜

더 많은 파워가 필요한 크리에이터를 위해:

- 더 높은 가입 보너스
- 월간 크레딧 지급 (영구)
- AI 이미지 생성
- 커스텀 도메인
- 비공개 미리보기 *(곧 제공 예정)*
- 우선 지원

*현재 플랜 및 가격은 [website2.ai/pricing](https://website2.ai/pricing)을 방문하세요.*

## 크레딧 관리

### 잔액 확인

1. 오른쪽 상단의 프로필 아이콘 클릭
2. **계정** 선택 또는 구독 모달 열기
3. **크레딧** 탭에서 현재 잔액 확인

### 거래 내역 보기

크레딧 탭에서 확인할 수 있는 것:

- **거래 유형** - 충전 (구매), 사용, 또는 보너스
- **금액** - 받은 크레딧은 양수, 사용한 크레딧은 음수
- **설명** - 크레딧이 무엇에 사용되었는지
- **날짜** - 거래가 발생한 시간

### 크레딧 갱신 일정

| 크레딧 유형 | 시점 |
|-------------|------|
| 일일 크레딧 | 매일 00:00 UTC |
| 월간 크레딧 (Pro) | 매월 1일 |
| 가입 보너스 | 계정 생성 시 일회성 |

**참고:** 일일 크레딧은 이월되지 않습니다. 사용하지 않은 일일 크레딧은 자정 UTC에 만료됩니다.

## 플랜 업그레이드

### 업그레이드 방법

1. 프로필 아이콘 클릭
2. **계정** → **구독** 탭 선택
3. 원하는 플랜 선택
4. Polar를 통해 결제 완료

### 업그레이드 시 발생하는 일

- **즉시:** 새 티어의 가입 보너스 크레딧 수령
- **매월 (Pro):** 1일에 월간 크레딧 수령

### 구독 관리

고객 포털에서 접근 가능:
- 결제 내역 보기
- 결제 방법 업데이트
- 청구서 다운로드
- 구독 취소

## 다운그레이드 및 취소

### 다운그레이드 시 발생하는 일

Pro에서 Free로 다운그레이드 시:

1. **커스텀 도메인 제거됨** - 연결된 모든 커스텀 도메인 연결 해제
2. **크레딧 할당 변경** - Free 티어 일일 크레딧 수령
3. **기능 제한됨** - Pro 기능 사용 불가

### 취소 정책

- 고객 포털에서 언제든지 취소
- 결제 기간 종료까지 접근 유지
- 기간 종료 후 계정이 Free로 다운그레이드

## 크레딧 사용 최적화 팁

크레딧을 더 효율적으로 사용하세요:

- **구체적으로** — 명확하고 상세한 프롬프트로 왔다 갔다 줄이기
- **집중적으로 변경** — 작고 단일한 변경은 대규모 재작성보다 적게 소비
- **일일 크레딧 사용** — 영구 크레딧 사용 전에 일일 크레딧으로 실험
- **현명하게 반복** — 불필요한 재생성을 최소화하도록 변경 계획

## FAQ

**구매한 크레딧은 만료되나요?**

구매한 크레딧과 가입 보너스 크레딧은 영구적이며 만료되지 않습니다. 월간 구독 크레딧은 미사용 시 1개월 이월됩니다. 일일 크레딧은 24시간 후 만료됩니다.

**생성 중 크레딧이 바닥나면 어떻게 되나요?**

현재 생성은 완료됩니다. 추가 변경을 하려면 크레딧을 추가해야 합니다.

**일일 크레딧은 왜 만료되나요?**

일일 크레딧은 정기적인 사용과 탐색을 장려합니다. 영구 크레딧에 추가되는 보너스입니다.

**크레딧 비용은 어떻게 계산되나요?**

크레딧은 약간의 마진이 포함된 AI 토큰 사용량을 기반으로 합니다. 더 복잡한 요청은 더 많은 AI 처리가 필요하므로 더 많은 크레딧이 소비됩니다.
