# 가격

## 구독 플랜

### Free 플랜

무료로 시작하세요:

- 가입 보너스 크레딧 (영구)
- 일일 크레딧 (24시간 만료)
- 공개 미리보기
- 기본 기능

### Pro 플랜

더 많은 파워가 필요한 크리에이터를 위해:

- 더 높은 가입 보너스
- 월간 크레딧 지급 (영구)
- AI 이미지 생성
- 커스텀 도메인
- 비공개 미리보기 *(곧 제공 예정)*
- 우선 지원

*현재 플랜 및 가격은 [website2.ai/pricing](https://website2.ai/pricing)을 방문하세요.*

## 플랜 업그레이드

### 업그레이드 방법

1. 프로필 아이콘 클릭
2. **계정** → **구독** 탭 선택
3. 원하는 플랜 선택
4. 결제 완료

### 업그레이드 시 발생하는 일

- **즉시:** 새 티어의 가입 보너스 크레딧 수령
- **매월 (Pro):** 1일에 월간 크레딧 수령

### 구독 관리

고객 포털에서 접근 가능:
- 결제 내역 보기
- 결제 방법 업데이트
- 청구서 다운로드
- 구독 취소

## 다운그레이드 및 취소

### 다운그레이드 시 발생하는 일

Pro에서 Free로 다운그레이드 시:

1. **커스텀 도메인 제거됨** - 연결된 모든 커스텀 도메인 연결 해제
2. **크레딧 할당 변경** - Free 티어 일일 크레딧 수령
3. **기능 제한됨** - Pro 기능 사용 불가

### 취소 정책

- 고객 포털에서 언제든지 취소
- 결제 기간 종료까지 접근 유지
- 기간 종료 후 계정이 Free로 다운그레이드
