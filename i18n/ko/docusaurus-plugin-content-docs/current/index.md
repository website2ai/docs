# Hando에 오신 것을 환영합니다

**아이디어만 말하세요. 웹사이트는 AI가 만듭니다.**

Hando는 자연어로 설명하면 프로덕션 수준의 웹사이트를 즉시 생성하는 AI 플랫폼입니다. 코딩 지식이 전혀 필요 없습니다.

## 왜 Hando인가요?

### 투명한 크레딧 시스템

:::tip 💰 사용한 만큼만 지불됩니다
- **복잡도 기반 과금** — 실제 작업 복잡도에 따라 크레딧이 차등 과금됩니다
- **사이트 생성 예시** — 약 13 크레딧 (이미지 미포함)
- **AI 이미지 생성** — 약 12 크레딧/장
- **초과 요금 없음** — 크레딧이 부족하면 작업이 멈출 뿐, 추가 청구되지 않습니다
- **호스팅 비용 포함** — 배포, CDN 비용이 모두 포함되어 있습니다
:::

### 최신 AI 모델

신규 AI 모델이 출시되면 빠르게 적용하여 제공합니다. 항상 최신 기술로 웹사이트를 생성하세요.

### 무료로 시작하세요

- 가입만 하면 **무료 크레딧** 지급
- 매일 **일일 크레딧** 충전
- 신용카드 등록 없이 시작 가능

자세한 내용은 [가격 페이지](/ko/pricing)를 확인하세요.

## 작동 방식

1. **설명하기** — AI에게 무엇을 만들고 싶은지 설명하세요
2. **생성하기** — 웹사이트가 실시간으로 생성되는 것을 지켜보세요
3. **다듬기** — 채팅으로 변경 및 개선 사항을 요청하세요
4. **배포하기** — 사이트가 즉시 라이브로 공유 가능합니다

## 주요 기능

### AI 기반 생성
일반 언어로 웹사이트를 설명하세요. AI가 의도, 스타일 선호도, 복잡한 요구 사항을 이해하고 React, TypeScript, Tailwind CSS를 사용하여 깔끔하고 현대적인 코드를 생성합니다.

### 실시간 미리보기
AI가 코드를 생성하는 동안 웹사이트가 만들어지는 것을 확인하세요. 빌드나 배포를 기다릴 필요 없이 작업하는 동안 실시간으로 볼 수 있습니다.

### 채팅 기반 편집
대화를 통해 웹사이트를 다듬으세요. "헤더를 고정으로 만들어줘" 또는 "문의 양식을 추가해줘"와 같은 변경 사항을 요청하면 즉시 업데이트됩니다.

### 버전 기록 (곧 제공 예정)
모든 변경 사항은 새 버전을 생성하게 됩니다. 기록을 탐색하고 한 번의 클릭으로 이전 버전을 복원할 수 있게 될 예정입니다.

### 즉시 배포
웹사이트가 글로벌 CDN에 자동으로 배포됩니다. 고유 URL을 즉시 공유하거나 자신의 도메인을 연결하세요.

### 프로덕션 수준 코드
생성된 웹사이트는 최신 기술을 사용합니다:
- TypeScript를 사용한 React 18+
- 스타일링을 위한 Tailwind CSS
- 빠른 빌드를 위한 Vite
- 기본적으로 모바일 반응형

### 커스텀 도메인
자신만의 도메인을 연결하세요. Pro 플랜에서 사용 가능합니다.

## 누구를 위한 것인가요?

- **창업가** — MVP와 랜딩 페이지를 빠르게 출시
- **중소기업** — 개발자 없이 전문적인 웹사이트 제작
- **디자이너** — 목업을 작동하는 프로토타입으로 변환
- **마케터** — 캠페인 페이지와 프로모션 사이트 제작
- **개발자** — 프로젝트 스캐폴딩 및 보일러플레이트 코드 생성
- **아이디어가 있는 모든 분** — 코딩 없이 비전을 실현

## 가격

Hando는 크레딧 기반 시스템을 사용합니다. 신규 사용자는 시작을 위한 무료 크레딧을 받습니다.

**플랜:**
- **Free** — 가입 보너스 + 일일 크레딧, 기본 기능
- **Pro** — 월간 크레딧, AI 이미지 생성, 커스텀 도메인

자세한 내용은 [가격](/ko/pricing)을 참조하거나 [hando.dev/pricing](https://hando.dev/pricing)을 방문하세요.
