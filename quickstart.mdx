## Get Started in 3 Steps

<Steps>
  <Step title="Sign Up">
    Go to [website2ai.com](https://website2ai.com) and click **"Sign in with Google"**

    Your account is created instantly - no email verification needed.
  </Step>

  <Step title="Open the Builder">
    Click **"Create New Project"** or navigate to the Builder
  </Step>

  <Step title="Describe Your Website">
    Tell the AI what you want to build and watch it create your website in real-time!
  </Step>
</Steps>

## Create Your First Website

### 1. Start a New Project

In the builder, you'll see a chat interface. Type your first prompt:

<Tip>
**Example prompts to try:**
- "Create a landing page for a productivity app"
- "Build a todo app with dark mode"
- "Make a portfolio website with a hero section and project gallery"
- "Create an e-commerce product page"
</Tip>

### 2. Watch AI Build Your Website

Once you hit enter, website2ai will:

<Steps>
  <Step title="Plan the Structure">
    AI analyzes your request and plans the file structure
  </Step>

  <Step title="Generate Code">
    Watch as files are created one by one in real-time
  </Step>

  <Step title="Live Preview">
    See your website appear in the preview panel as it's built
  </Step>
</Steps>

This typically takes **30-60 seconds** depending on complexity.

### 3. Preview Your Website

Your website appears in the **live preview panel** on the right side:

- ✅ Fully interactive and functional
- ✅ Responsive design
- ✅ All components working

### 4. Make Changes

Want to modify your website? Just chat with the AI:

```
"Add a contact form"
"Change the colors to blue"
"Make the hero section taller"
"Add a dark mode toggle"
```

Each change creates a new version - you can always go back!

### 5. Download Your Code

Click the **"Download"** button to get:

- Complete React project with TypeScript
- All source files
- package.json with dependencies
- Ready to run locally or deploy

  <Accordion title="Be Specific">

</AccordionGroup>

## Understanding the Interface

### Left Panel: Chat & Code

- **Chat Input**: Describe what you want or request changes
- **File Tree**: Browse all generated files
- **Code Editor**: View and copy any file's code

### Right Panel: Live Preview

- **Preview Window**: See your website running live
- **Responsive Toggle**: Test mobile/tablet/desktop views
- **Refresh**: Reload the preview if needed

### Top Bar

- **Project Title**: Click to rename
- **Version Dropdown**: Switch between different versions
- **Download Button**: Get your complete React project

## Managing Projects

### View All Projects

Click **"My Projects"** to see all your websites:

- See project thumbnails and titles
- Filter by status (completed, in progress)
- Sort by date created or updated

### Version History

Every change creates a new version:

1. Click the **version dropdown**
2. See all versions with their prompts
3. Click any version to view or restore it

<Tip>
Versions are numbered automatically: v1, v2, v3...

Each version is a complete snapshot - you can always go back!
</Tip>

Let's walk through creating a complete landing page:


## Common Questions

    Most websites generate in **30-90 seconds**:
    - Simple pages: ~30 seconds
    - Medium complexity: ~45 seconds
    - Complex apps: ~60-90 seconds
  </Accordion>

  <Accordion title="Can I edit the code directly?">
    You can view and copy code from any file. To make changes, chat with the AI or download the project and edit locally.

  <Accordion title="What happens to old versions?">
    All versions are saved automatically. You can view, restore, or branch from any previous version.
  </Accordion>

  <Accordion title="Can I use this commercially?">
    Yes! All generated code is yours to use however you want - including commercial projects.
  </Accordion>
</AccordionGroup>
